Director

an R package for dynamic, browser-enabled visualization of multi-level regulatory interactions

Summary

As datasets from high-throughput technology grow increasingly complex, novel approaches are needed to visually convey key findings on multiple, related-levels of data quickly and effectively. Director is an R package designed to streamline the visualization of regulatory, downstream effect in a biological network. It utilizes a customized Sankey plugin of the JavaScript library D3 to provide a fast and easy, browser-enabled solution to discovering potentially interesting molecular interactions that would not otherwise be evident using standard network visualization or statistical methods alone. The diagrams are robust, interactive, and packaged as highly-portable HTML files that eliminate the need for third-party software to view.

Read more with a basic walkthrough in the package vignette.

Installation

Install Director from its GitHub repository. In R, you first need to install devtools and ensure Bioconductor is a named source.

install.packages("devtools")
source("https://bioconductor.org/biocLite.R")`

Then using the 'install_github' function in the devtools package:

library(devtools)
install_github("kzouchka/Director")

Example use

The following test case draws a set of regulating miRNAs and their target genes in key biological pathways using default package settings.

library(Director)
data(ovca) # import data
initSankey() # same as initSankey(directory=getwd())
Level1 <- createList(poorprog$Level1) # miRNA-gene data
Level2 <- createList(poorprog$Level2) # gene-pathway data.
List <- append2List(Level1, Level2) # append lists.

Sankey <- makeSankey(List, averagePath = TRUE) # calculate colour values
Sankeyd <- drawSankey(Sankey,caption="Regulatory pathway",
width=1400, height=900) # raw HTML generated
writeSankey(name="FDRpathways") # write to working directory as ’FDRpathways.html’

# Use the following to relaunch the diagram as a web page
library(htmltools)
html_print(Sankeyd)
